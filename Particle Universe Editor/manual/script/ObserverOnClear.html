On Clear Observer

Observes whether there are no particles in the scene anymore. It concerns the particles emitted by emitters that belong to the same technique as where the OnClear observer is registered. Observing specific particle types has no meaning.

Note, not all eventhandlers are useful to be registered at this observer. Most of the eventhandlers expect a specific particle, but individual particles are not observed (only the absence of particles in the scene is observed). Using these type of eventhandlers will result in an exception situation.
The DoEnableComponent and the DoStopSystem are typical eventhandlers that can be used in a combination with the OnClearObserver.

Definition

Syntax:

observer   OnClear   [observerName]   {observer properties}

See ParticleObserver for explanation of the syntax.

Example: 

observer                OnClear
{
   ...
}

Properties

This Observer doesn't have own properties.
