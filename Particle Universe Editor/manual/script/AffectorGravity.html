Gravity Affector

The Gravity Affector attracts or rejects particles, using Newton's law of universal gravitation. The position of the affector is important. This forms the center of gravity.

Definition

Syntax:

affector   Gravity   [affectorName]   {affector properties}

See ParticleAffector for explanation of the syntax.

Example: 

affector                   Gravity
{
   position                0 500 0
   gravity                 1000
}

Properties


Gravity

Defines the gravity of this affector. The gravity causes particles to be attracted (or rejected if negative). It uses Newton's law of universal gravitation. Syntax:

gravity   #value

gravity is the keyword.
#value is a real value that defines the gravity. Default: 1.0.
