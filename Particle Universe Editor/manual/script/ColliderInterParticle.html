Inter Particle Collider

The Inter Particle Collider is used to define that particles are colliding with each other. It uses a bounding sphere collision detection, while different types of collision response can be used. This affector make use of the Spatial Hashing function of the Particle Technique. Make sure that the Spatial Hashing attributes of the Particle Technique have been set to a proper value.

Beware that inter particle collision is part of Particle Universe itself. If you want more sophisticated collision and responses, Particle Universe also has the option to integrate with NvidiaÂ´s PhysX engine, formerly owned by Ageia.

Definition

Syntax:

affector   InterParticleCollider   [affectorName]   {affector properties}

See ParticleAffector for explanation of the syntax.

Example: 

affector                     InterParticleCollider
{
    adjustment               0.8
    collision_response       angle_based_velocity
}

Properties


Adjustment

The adjustment property is used to adjust the collision detection. For example, in case where a texture doesn't completely cover a billboard at the edges, it might look as if particles are colliding with the surrounding space. If the adjustment property is set to a value < 1, the particles are able to approach each other closer, before the collision detection is activated. Syntax:

adjustment   #value

adjustment is the keyword. ip_adjustment is deprecated from version 1.2
#value is a value > 0 and defines the amount of adjustment needed to correct the collision detection. Default: 1.0.

Collision Response

Defines how a particle behaves after it collides with another particle. Syntax:

collision_response   average_velocity | angle_based_velocity

collision_response is the keyword. ip_collision_response is deprecated from version 1.2
average_velocity means that the velocity of each particle after collision is the average of the two velocities before collision. This is the default.
angle_based_velocity means that the velocity after collision is based on the angle between the direction vector before and the direction vector after collision.
