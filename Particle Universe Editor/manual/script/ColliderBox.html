Box Collider

The Box Collider represents a box shape with which the particles collide.

Beware that the Box Collider is part of Particle Universe itself. If you want more sophisticated collision and responses, Particle Universe also has the option to integrate with NvidiaÂ´s PhysX engine, formerly owned by Ageia.

Definition

Syntax:

affector   BoxCollider   [affectorName]   {affector properties}

See ParticleAffector for explanation of the syntax.

Example: 

affector                     BoxCollider
{
    position                 0 -100 0
    box_width                20
    box_height               300
    box_depth                20
    intersection             point
    collision_type           flow
}

Properties


Box Width

Defines the width of the box collider. Syntax:

box_width   #value

box_width is the keyword. box_collider_width is deprecated from version 1.2
#value is a real value > 0 that defines the width. Default: 0.

Box Height

Defines the height of the box collider. Syntax:

box_height   #value

box_height is the keyword. box_collider_height is deprecated from version 1.2
#value is a real value > 0 that defines the height. Default: 0.

Box Depth

Defines the depth of the box collider. Syntax:

box_depth   #value

box_depth is the keyword. box_collider_depth is deprecated from version 1.2
#value is a real value > 0 that defines the depth. Default: 0.

Inner Collision

Defines whether the collision takes place inside of the box, or outside. If the value is set to true, particles collide with the inner sides of the box. Syntax:

inner_collision   true | false

inner_collision is the keyword.
true | false defines wither the collision is inside of the box. Default: false.
