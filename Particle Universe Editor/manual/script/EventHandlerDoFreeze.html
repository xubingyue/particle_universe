Do Freeze Event Handler

Freezes a particle. This means that the particle doesn't move anymore, but it will still be affected by the affectors (except for movement).

Definition

Syntax:

handler   DoFreeze    [handlerName]   {}

See ParticleEventHandler for explanation of the syntax.

Example: 

observer                OnCollision
{
   ...
   handler              DoFreeze {}
}

Properties

This Event Handler doesn't have properties.
