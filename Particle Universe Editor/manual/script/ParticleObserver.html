Particle Observer

The syntax of a Particle Observer definition is:

observer   observerType   [observerName]   {observer properties}

observer is the keyword.
observerType is the type of observer.
observerName defines the name of the observer. The observerName is optional but required if the observer must be identified.
The observer properties are handled in the paragraph 'Base Properties'.

Example: 

observer                   OnPosition
{
   ...
}

Base Properties

The Base Properties apply to all observers.

Enabled

Enables or disables the observer as soon as the particle system starts. This makes it possible not to observe the particles immediately, but for instance if another event occurs. Syntax:

enabled   true | false

enabled is the keyword.
true | false defines whether the observer is enabled (true) or disabled (false). Default: true

Observe Particle Type

Besides emission of visual particles, also other particle types can be emitted. The observer observes all particles by default, but if specific particle types must be observed, this property is used to define the particle type that should be observed only. Syntax:

observe_particle_type   visual_particle | emitter_particle | technique_particle | affector_particle | system_particle

observe_particle_type is the keyword.
visual_particle | emitter_particle | technique_particle | affector_particle | system_particle. Set one of these 5 values to define which particle type is observed. Default: visual_particle.

Observe Interval

Only observe after every interval. This prevents continuous observation. Syntax:

observe_interval   #value

observe_interval is the keyword.
#value. defines the interval after which the observer is active. Default: No interval set.

Observe Until Event

This attribute makes it possible to stop observing after the event handler(s) are called for the first time. This means that the event handlers are called only once and after that, the observer stops observing. Syntax:

observe_until_event   true | false

observe_until_event is the keyword.
true | false defines whether the observer should stop observing after the event handler(s) are called for the first time (= true). Default: false.

Event Handler

Invokes an Event Handler is the condition of the Observer is true. See Event Handler for the syntax.
