Particle Behaviour

Definition

The syntax of a Particle Behaviour definition is:

behaviour   behaviourType   {behaviour properties}

behaviour is the keyword.
behaviourType is the type of behaviour.
The behaviour properties are handled in the paragraph 'Properties' of each specific Behaviour.

Example: 

behaviour                 Slave
{
}

Base Properties

Particle Behaviour doesn't have base properties.
