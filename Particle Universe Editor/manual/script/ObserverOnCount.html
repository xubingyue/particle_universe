On Count Observer

Counts the number of emitted particles. If count validation is true (i.e. count_threshold greater_than 100), the registered event handlers are called.

Definition

Syntax:

observer   OnCount   [observerName]   {observer properties}

See ParticleObserver for explanation of the syntax.

Example: 

observer                OnCount
{
   count_threshold      greater_than 100
   handler              DoExpire {}
}

Properties


Count Threshold

Set the lower/upper threshold value of the number of emitted particles. If the number of emitted particles exceeds this value, the event handlers are triggered. Syntax:

count_threshold   less_than | greater_than | equals   #value

count_threshold is the keyword.
less_than | greater_than | equals defines the comparision operator. Default: less_than
#value is the threshold value that determines when the event handlers are triggered. Default: 0
