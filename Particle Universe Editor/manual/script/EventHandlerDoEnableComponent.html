Do Enable Component Event Handler

Enable or disable a specific Emitter, Affector or Technique, identified by its name.

Definition

Syntax:

handler   DoEnableComponent   [handlerName]   {handler properties}

See ParticleEventHandler for explanation of the syntax.

Example: 

handler                 DoEnableComponent
{
    enable_component    emitter_component    myEmitter    false
}

Properties


Component To enable/disable

Define the component type (emitter, affector or technique) and name of the component that must be enabled or disabled. Syntax:

enable_component   emitter_component | affector_component | technique_component | observer_component   #name   true|false

enable_component is the keyword.
emitter_component | affector_component | technique_component | observer_component defines whether the component to enable/disable is an emitter, an affector, a technique or an observer.
#name is the name of the component.
true | false means that the component is enabled (true) or disabled.
