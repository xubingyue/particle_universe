Path Follower

The Path Follower makes it possible to define a path over which the particle moves. By adding points to the Path Follower, the path is calculated by interpolation between the points. The path is defined for the lifetime of the particle, so if the particles' time to live is decreased, it it still follows the whole path.

Definition

Syntax:

affector   PathFollower   [affectorName]   {affector properties}

See ParticleAffector for explanation of the syntax.

Example: 

affector                    PathFollower
{
    path_follower_point     -40 -10 -40
    path_follower_point     -10 50 40
    path_follower_point     -10 -20 -10
    path_follower_point     40 -50 70
    path_follower_point     -100 -10 -60
}

Properties


Control Points (Add Position)

The points added to this affector are used to calculate the path, by using smooth interpolation. Syntax:

path_follower_point   #x   #y   #z

path_follower_point is the keyword. Multiple 'path_follower_point' attributes can be added.
#x defines the x value of the point.
#y defines the y value of the point.
#z defines the z value of the point.
