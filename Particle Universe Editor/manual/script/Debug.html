Debug log

Particle Universe contains an option that analyses the Particle System at runtime and writes the results in the Ogre.log as soon as the Particle System stops. This results in an overview of the maximum number of emitted particles (for all particle types). With these numbers, all particle quota (visual_particle_quota, emitted_emitter_quota, ...) can be set properly, so no more memory is allocated than needed. Enabling this feature is done by uncommenting the line //#define PU_LOG_DEBUG from ParticleUniversePrerequisites.h
