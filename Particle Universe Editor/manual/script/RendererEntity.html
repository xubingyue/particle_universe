Entity Renderer

Renders particles as an entity, using the material set in the technique.

Definition

Syntax:

renderer   Entity   {renderer properties}

See ParticleRenderer for explanation of the syntax.

Example: 

technique
{
    renderer                          Entity
    {
        mesh_name                     pu_stone.mesh
    }
}

Properties

Mesh Name

Defines the name of the mesh that is rendered (as an entity). Syntax:

mesh_name   #name

mesh_name is the keyword. entity_renderer_mesh_name is deprecated from version 1.2
#name is the name of the mesh.

Orientation Type

With this setting, the orientation of the entity particle can be changed. By default, the orientation of the particle itself is used, but that can be overruled eith other settings, for example aligning the particle towards its direction. Syntax:

entity_orientation_type   ent_oriented_self | ent_oriented_self_mirrored | ent_oriented_shape

entity_orientation_type is the keyword.
ent_oriented_self means that the entity particle is aligned towards its direction. This setting overrules the orientation of the particle itself.
ent_oriented_self_mirrored means that the entity particle is aligned towards its direction, but the model is mirrored. This setting overrules the orientation of the particle itself.
ent_oriented_shape means that the orientation of the particle is taken into account. This is the default setting.
