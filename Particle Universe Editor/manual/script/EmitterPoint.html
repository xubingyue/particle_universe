Point Emitter

Emits particles from a point.

Definition

Syntax:

emitter   Point   [emitterName]   {emitter properties}

See ParticleEmitter for explanation of the syntax

Example: 

emitter               Point
{
   ...
}

Properties

This emitter doesn't have its own properties. Only the base properties of a Particle Emitter apply to this emitter.
