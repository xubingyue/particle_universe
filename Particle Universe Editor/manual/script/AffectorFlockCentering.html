Flock Centering Affector

This affector is one element of the Flocking behaviour algorithm. It determines the 'center' of all particles in a technique and adjusts the particle direction, so the particle is steering towards the center.

Definition

Syntax:

affector   FlockCentering   [affectorName]   {affector properties}

See ParticleAffector for explanation of the syntax.

Example: 

affector                        FlockCentering
{
}

Properties


This Affector doesn't have any properties.
