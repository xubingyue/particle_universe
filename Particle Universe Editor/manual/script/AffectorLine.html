Line Affector

The Line Affector is typically used in cooperation with the line emitter. Where the line emitter emits particles on the line or at some distance from the line, the line affector updates the particle position. This position however, is still near the given line. The update frequency and the matter of movement can both be defined.

Definition

Syntax:

affector   Line   [affectorName]   {affector properties}

See ParticleAffector for explanation of the syntax.

Example: 

affector                     Line
{
   max_deviation             140
   time_step                 0.03
   end                       1500 300 0
   drift                     0.3
}

Properties


Line end (End position)

Defines the direction and length of the line by means of a 3d vector. Syntax:

end   #x   #y   #z

end is the keyword. line_aff_end is deprecated from version 1.2
#x is the x value of the vector. Default: 0.
#y is the y value of the vector. Default: 0.
#z is the z value of the vector. Default: 0.

Maximum Deviation

Setting the maximum deviation, which defines how far (max.) from the line a particle is repositioned. Syntax:

max_deviation   #value

max_deviation is the keyword. line_aff_max_deviation is deprecated from version 1.2
#value is the maximum distance from the line. Default: 0.

Time Step

Defines the update time. The shorter the period, the faster a particle will be repositioned. Very short time steps are good to produce electricity type of behaviour. Syntax:

time_step   #value

time_step is the keyword. line_aff_time_step is deprecated from version 1.2
#value is the update time. Default: 0.0.

Drift

With the drift attribute it is possible to define whether the particles' new position is near the previous position or (when the drift value is very high), that the new position is generated totally random. Syntax:

drift   #value

drift is the keyword. line_aff_drift is deprecated from version 1.2
#value is a real value between [0..1]. A lower value means that the new particle position is near the previous one. Default: 0.0.
