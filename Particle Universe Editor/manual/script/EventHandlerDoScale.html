Do Scale Event Handler

Scale one of the particles' attributes.

Definition

Syntax:

handler   DoScale   [handlerName]   {handler properties}

See ParticleEventHandler for explanation of the syntax.

Example: 

handler                 DoScale
{
    scale_fraction      0.5
    scale_type          st_time_to_live
}

Properties


Scale Fraction

The fraction with which the particle attribute is scaled. The scaling is done by means of scaleFraction * timeSinceLastUpdate * attributeValue. Syntax:

scale_fraction   #value

scale_fraction is the keyword.
#value is the value of the scale fraction. Default: 0.2

Scale Type

The particle attribute that is scaled. Syntax:

scale_type    time_to_live | velocity

scale_type is the keyword.
time_to_live | velocity scales either the time_to_live or the particle velocity. Default: time_to_live sc_time_to_liveand sc_velocity are deprecated from version 1.2
