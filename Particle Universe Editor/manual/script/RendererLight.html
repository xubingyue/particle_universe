Light Renderer

Renders particles as a light, which enables dynamic lightning. One example is a fire particle system with an additional light particle that dynamically lights the scene. Be careful with the number of lights that are emitted, because it slows down the framerate considerably. Set the visual_particle_quota very low.

Although it concerns a light renderer, it is not possible to set the diffuse colour (although it possible to set the specular colour). This is because the light renderer 'inherits' its colour from the particle. This makes it possible to change the colour of the light similar as a regular particle, by means of a Colour Affector, for example.

Definition

Syntax:

renderer   Light   {renderer properties}

See ParticleRenderer for explanation of the syntax.

Example: 

technique
{
    renderer                          Light
    {
        light_renderer_light_type     point
        light_renderer_diffuse        1 0 0
        light_renderer_specular       1 1 0
        light_renderer_att_range      200
    }
}

Properties

Light Type

This is either a spotlight or a pointlight. Directional lights are not supported. Syntax:

light_renderer_light_type   point | spot

light_renderer_light_type is the keyword.
point | spot define the type of light. Default: point

Specular Colour

The colour of the specular light. Syntax:

light_renderer_specular   #r #g #b

light_renderer_specular is the keyword.
#r is the red-value. This is a value between 0..1
#g is the green-value. This is a value between 0..1
#b is the blue value. This is a value between 0..1. Default rgb values: 0 0 0

Attenuation Range

Attenuation range defines the upper range where the light is still active. The range value is in world units Syntax:

light_renderer_att_range   #value

light_renderer_att_range is the keyword.
#value is a value > 0. Default: 100000

Attenuation Constant

This is the 'constant' factor in the attenuation formula. Syntax:

light_renderer_att_constant   #value

light_renderer_att_constant is the keyword.
#value is a value between 0..1. Default: 1 (no attenuation)

Attenuation Linear

Defines whether attenuation is performed gradually in an even manner. Syntax:

light_renderer_att_linear   #value

light_renderer_att_linear is the keyword.
#value is a value > 0. Default: 0

Attenuation Quadratic

Defines whether attenuation is performed gradually in a curved manner. Syntax:

light_renderer_att_quadratic   #value

light_renderer_att_quadratic is the keyword.
#value is a value > 0. Default: 0

Spotlight Inner Angle

The angle of the inner cone of a spotlight. Syntax:

light_renderer_spot_inner   #value

light_renderer_spot_inner is the keyword.
#value is a value > 0. Default: 30

Spotlight Outer Angle

The angle of the outer cone of a spotlight. Syntax:

light_renderer_spot_outer   #value

light_renderer_spot_outer is the keyword.
#value is a value > 0. Default: 40

Falloff

The rate of falloff between the inner and outer cones of a spotlight. Syntax:

light_renderer_falloff   #value

light_renderer_falloff is the keyword.
#value is a value > 0. Default: 1

Power Scale

Scaling factor for the relative power of a light (for HDR). Syntax:

light_renderer_powerscale   #value

light_renderer_powerscale is the keyword.
#value is a value > 0. Default: 1

Flash Frequency

This makes it possible to flash the light on/off with a certain frequency. Syntax:

flash_frequency   #value

flash_frequency is the keyword.
#value defines the frequency of the flash. Default: 0

Flash Length

Only in case the Flash Frequency is set, this property has a meaning. It defines how long the light is 'on'. Syntax:

flash_length   #value

flash_length is the keyword.
#value defines the how long the light is 'on'. Default: 0

Flash Random

Where a combination of Flash Frequency and Flash Lenght causes a steady flashing effect, does the combination of Flash Frequency and Flash Random result in an irregular flashing effect. Syntax:

flash_random   true | false

flash_random is the keyword.
true | false indicates whether the flash effect is regular (false) or irregular (true). Default: false
