On Random Observer

Calls the registered eventhandlers when a randomly generated value exceeds a certain threshold.

Definition

Syntax:

observer   OnRandom   [observerName]   {observer properties}

See ParticleObserver for explanation of the syntax.

Example: 

observer                OnRandom
{
   random_threshold     0.6
   handler              DoExpire {}
}

Properties


Random Threshold

Set the threshold value. The range is between [0..1]. If a generated random value - which also has a range between [0..1] - is larger than the threshold, the registered eventhandlers are called. The random value is by default generated in each update cycle of the Particle System. Note, that this frequency can be changed by setting the 'observe_interval'. Syntax:

random_threshold   #value

random_threshold is the keyword.
#value is the threshold value. If the generated random value > #value, the registered eventhandlers are called. Default: 0.5
