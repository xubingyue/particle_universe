Box Emitter

Emits particles from a box shape.

Definition

Syntax:

emitter   Box   [emitterName]   {emitter properties}

See ParticleEmitter for explanation of the syntax

Example: 

emitter               Box
{
   ...
   box_width          200
   box_height         10
   box_depth          200
}

Properties


Box Width

Defines the width of the box in which particles are emitted. Syntax:

box_width   #value

box_width is the keyword. box_em_width is deprecated from version 1.2
#value is a real value > 0 that defines the width. Default: 100.

Box Height

Defines the height of the box in which particles are emitted. Syntax:

box_height   #value

box_height is the keyword. box_em_height is deprecated from version 1.2
#value is a real value > 0 that defines the height. Default: 100.

Box Depth

Defines the depth of the box in which particles are emitted. Syntax:

box_depth   #value

box_depth is the keyword. box_em_depth is deprecated from version 1.2
#value is a real value > 0 that defines the depth. Default: 100.
