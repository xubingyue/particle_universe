Vortex Extern

The Vortex Extern is a wrapper around the Vortex Affector. Its capabilities are extended by making the Vortex Affector 'attachable'. This means that it can be attached to another SceneNode, but is still able to affect the particles in the Particle Technique where this Vortex Extern is registered. Although the registration of this Extern to a Particle Technique can be done by script, attaching it to a SceneNode must still be done programmatically. See below for an example. The benefit of having a Vortex Extern is, that you are able to affect particles from outside the Particle System. Effects like a helicopter that blows away leaves when it approaches can be created easily now.

Definition

Syntax:

extern   Vortex   [externName]   {extern properties}

See Extern for explanation of the syntax.

Script example: 

system with_an_extern
{
    technique
    {
        ....
        extern                                Vortex
        {
            rotation_axis                     0 1 0
            rotation_speed                    5
            distance_threshold                300
        }
    }
}

Additional code example to attach the extern to a SceneNode (sys is a ParticleSystem): 

ParticleUniverse::ParticleSystem* sys = 
   ParticleUniverse::ParticleSystemManager::getSingleton().createParticleSystem
   ("mySystem", "with_an_extern", mSceneMgr);
SceneNode* externNode = mSceneMgr->getRootSceneNode()->createChildSceneNode();
Attachable* attachable = static_cast<Attachable*>(sys->getTechnique(0)->getExtern(0));
if (attachable)
{
    externNode->attachObject(attachable);
    externNode->setPosition(Vector3(-300, 100, -100));
}

Properties


Distance Threshold

Maximum distance where the Vortex Extern doesn't affect the particle anymore. Syntax:

distance_threshold   #value

distance_threshold is the keyword. attachable_distance_threshold is deprecated from version 1.2
#value is a real value that defines the threshold speed. Default: Infinite.
