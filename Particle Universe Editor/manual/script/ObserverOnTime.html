On Time Observer

Observes the time since the start of the Particle System or the time since the particle was emitted. If the time is equal, less or greater than a certain threshold value, the registered Event Handlers are called.

Definition

Syntax:

observer   OnTime   [observerName]   {observer properties}

See ParticleObserver for explanation of the syntax.

Example: 

observer                   OnTime
{
   on_time                 greater_than 4
   handler                 DoEnableComponent
   {
       enable_component    emitter_component    myEmitter    false
   }
}

Properties


Time Threshold

Defines the threshold time and calls the registered eventhandlers if the time is before or after that value. Syntax:

on_time   less_than | greater_than | equals   #value

on_time is the keyword.
less_than | greater_than | equals defines the comparision operator. Default: greater_than
#value is the threshold value that determines when the event handlers are triggered. Default: 0

Since Start System

Since Start System is an indication that determines whether the time threshold condition refers to the time since the start of the Particle System, or the time since the particle was emitted. Syntax:

since_start_system   true | false

since_start_system is the keyword.
true | false defines whether the time condition refers to the time since the Particle System was started (true) or the time since the particle was emitted. Default: false.
