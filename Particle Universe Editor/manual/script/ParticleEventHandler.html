Particle Event Handler

Definition

The syntax of a Particle EventHandler definition is:

handler   handlerType   [handlerName]   {handler properties}

handler is the keyword.
handlerType is the type of handler.
handlerName defines the name of the event handler. The handlerName is optional but required if the handler must be identified.
The handler properties are handled in the paragraph 'Base Properties'.

Example: 

observer                   OnPosition
{
   ...
   handler                 DoExpire {}
}

Base Properties
