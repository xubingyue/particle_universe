Ribbon Trail Renderer

Renders particles as ribbontrails.

Definition

Syntax:

renderer   RibbonTrail   {renderer properties}

See ParticleRenderer for explanation of the syntax.

Example: 

technique
{
    renderer                                RibbonTrail
    {
        use_vertex_colours                  true
        max_elements                        60
        ribbontrail_length                  600
        ribbontrail_width                   10
        random_initial_colour               true
        colour_change                       0.8 0.8 0.8 0.8
    }
}

Properties

Use Vertex Colours

Set indication whether to use vertex colours or not. If set to true, use the Colour properties to set initial colours and colour changes . Syntax:

use_vertex_colours   true | false

use_vertex_colours is the keyword. ribbontrail_vertex_colours is deprecated from version 1.2
true | false If set to true, vertex colours are used. Default: true

Max Chain Elements

Maximum number of elements in a chain. The more elements, the smoother the trails. Syntax:

max_elements   #value

max_elements is the keyword. ribbontrail_max_elements is deprecated from version 1.2
#value Maximum of chain elements. Default: 10

Trail Length

The length of the ribbontrail. If the length is not defined, the default length of the particle is used (default length is defined in the technique). Syntax:

ribbontrail_length   #value

ribbontrail_length is the keyword.
#value defines the length of the ribbontrail. Default: 400

Trail Width

The width of the ribbontrail. If the width is not defined, the default width of the particle is used (default width is defined in the technique). Syntax:

ribbontrail_width   #value

ribbontrail_width is the keyword.
#value defines the width of the ribbontrail. Default: 5

Random Initial Colour

Determine whether the initial colour of all chains is set manually or whether the colour of each chain is randomised. This value is only meaningfull if 'ribbontrail_vertex_colours' is set to true. Syntax:

random_initial_colour   true | false

random_initial_colour is the keyword. ribbontrail_random_initial_colour is deprecated from version 1.2
true | false If set to true, initial colours are randomised. Default: true .

Initial Colour

Set the starting colour of all ribbontrails. This value is only meaningfull if 'ribbontrail_vertex_colours' is set to true. Syntax:

initial_colour   #red   #green   #blue   [#alpha]

initial_colour is the keyword. ribbontrail_initial_colour is deprecated from version 1.2
#red   #green   #blue   [#alpha] defines the initial colour. Default: White

Colour Change

Define the colourvalue that is subtracted from the ribbontrail colour each second. This value is only meaningfull if 'ribbontrail_vertex_colours' is set to true. Syntax:

colour_change   #red   #green   #blue   [#alpha]

colour_change is the keyword. ribbontrail_colour_change is deprecated from version 1.2
#red   #green   #blue   [#alpha] defines the colour change. Default: 0.5 0.5 0.5 0.5
