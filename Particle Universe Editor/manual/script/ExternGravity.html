Gravity Extern

The Gravity Extern is a wrapper around the Gravity Affector. Its capabilities are extended by making the Gravity Affector 'attachable'. This means that it can be attached to another SceneNode, but is still able to affect the particles in the Particle Technique where this Gravity Extern is registered. Although the registration of this Extern to a Particle Technique can be done by script, attaching it to a SceneNode must still be done programmatically. See below for an example. The benefit of having a GravityExtern is, that you are able to affect particles from outside the Particle System. Effects like a helicopter that blows away leaves when it approaches can be created easily now.

Definition

Syntax:

extern   Gravity   [externName]   {extern properties}

See Extern for explanation of the syntax.

Script example: 

system with_an_extern
{
    technique
    {
        ....
        extern                   Gravity
        {
            gravity              1000
        }
    }
}

Additional code example to attach the extern to a SceneNode: 

SceneNode* externNode = mSceneMgr->getRootSceneNode()->createChildSceneNode();
Attachable* attachable = static_cast<Attachable*>(sys->getTechnique(0)->getExtern(0));
if (attachable)
{
    externNode->attachObject(attachable);
    externNode->setPosition(Vector3(-300, 100, -100));
}

Properties


Distance Threshold

Maximum distance where the Gravity Extern doesn't affect the particles anymore. Syntax:

distance_threshold   #value

distance_threshold is the keyword. attachable_distance_threshold is deprecated from version 1.2
#value is a real value that defines the threshold speed. Default: Infinite.

Gravity

Defines the gravity of this extern. The gravity causes particles to be attracted (or rejected if negative). It uses Newton's law of universal gravitation. Syntax:

gravity   #value

gravity is the keyword.
#value is a real value that defines the gravity. Default: 1.0.
