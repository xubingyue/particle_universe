Sphere Collider Extern

The Sphere Collider Extern is a wrapper around the Sphere Collider. Its capabilities are extended by making the Sphere Collider 'attachable'. This means that it can be attached to another SceneNode, but is still able to affect the particles in the Particle Technique where this Sphere Collider Extern is registered. Although the registration of this Extern to a Particle Technique can be done by script, attaching it to a SceneNode must still be done programmatically. See below for an example. The benefit of having a Sphere Collider Extern is, that you are able to affect particles from outside the Particle System.

Definition

Syntax:

extern   SphereCollider   [externName]   {extern properties}

See Extern for explanation of the syntax.

Script example: 

system with_an_extern
{
    technique
    {
        ....
        extern                        SphereCollider collider_1
        {
            radius                    200
            intersection              point
            collision_type            bounce
            bouncyness                1
        }
        extern                        SphereCollider collider_2
        {
            radius                    100
            intersection              point
            collision_type            flow
        }
    }
}

Additional code example to attach the extern to a SceneNode: 

SceneNode* externNode = mSceneMgr->getRootSceneNode()->createChildSceneNode();
Attachable* attachable = static_cast<Attachable*>(sys->getTechnique(0)->getExtern("collider_2"));
if (attachable)
{
    externNode->attachObject(attachable);
    externNode->setPosition(Vector3(-300, 100, -100));
}

Properties


Distance Threshold

Maximum distance where the collision detection of the Extern Sphere Collider isn't performed anymore. Syntax:

distance_threshold   #value

distance_threshold is the keyword. attachable_distance_threshold is deprecated from version 1.2
#value is a real value that defines the threshold speed. Default: Infinite.
