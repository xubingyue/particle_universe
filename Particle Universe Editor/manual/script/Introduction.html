Introduction

Why another particle plugin for Ogre?

Well, although the current ParticleFX plugin in Ogre works good for standard particle systems, there are a number of situations where it is insufficient. For instance, an explosion with smoke, fire and debris with an emission rate that starts high and gradually decreases is not possible (not without additional code at least) using the ParticleFX plugin. The Particle Universe plugin makes these kinds of effects possible, all defined within one script.

The Particle Universe plugin is loosely based on Ogre's ParticleFX plugin, but built from the ground up, with more features and improved scripting possibilities.
