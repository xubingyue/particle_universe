Sphere Surface Emitter

Emits particles on the surface of a sphere. Note, that the auto_direction has a meaning in this emitter. If auto_direction is set to true, the particles are emitted perpendicular on the spheres' surface. The 'angle' property provides a variation on this direction.

Definition

Syntax:

emitter   SphereSurface   [emitterName]   {emitter properties}

See ParticleEmitter for explanation of the syntax.

Example: 

emitter                  SphereSurface
{
   ...
   radius                 200
}

Properties


Radius

Defines the radius of the sphere from which particles are emitted. Syntax:

radius   #value

radius is the keyword. sphere_surface_em_radius is deprecated from version 1.2
#value is a real value > 0 that defines the radius of the sphere. Default: 10.
