*.pua script

The Particle Universe plugin distinguishes 2 types of scripts *.pua and *.pu scripts, which are also compiled in this order. *.pua stands for Particle Universe Alias script and is particularly used to define pieces of scripts that can be reused in other scripts. These reusable scripts are identified by means of a unique aliasname.

Aliasses can be defined for the following components:

  • Renderers
  • Techniques
  • Emitters
  • Affectors
  • Observers
  • Handlers
  • Behaviour

The syntax of an alias is:

alias   aliasname   {componentdefinition}

Example:

alias                             myEmitterAlias
{
   emitter                        Point
   {
      emission_rate               40
      angle                       40
      direction                   0 1 0
      velocity                    100
      time_to_live                dyn_random
      {
         min                      2
         max                      6
      }
   }
}

The component definition is the same as in a regular particle script (*.pu). So, instead of using the whole emitter definition in multiple Particle Systems, we simply refer to it by means of its alias name 'myEmitterAlias'.
