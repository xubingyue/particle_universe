Circle Emitter

Emits particles on a circle shape. The 'auto_direction' property moves the particles outside from the center.

Definition

Syntax:

emitter   Circle   [emitterName]   {emitter properties}

See ParticleEmitter for explanation of the syntax

Example: 

emitter                Circle
{
   ...
   radius              100
   emit_random         false
   step                0.2
}

Properties


Radius

The radius of the circle. Syntax:

radius   #value

radius is the keyword. circle_em_radius is deprecated from version 1.2
#value is a real value > 0 that defines the radius of the circle. Default: 100.

Step

The step value determines the distance between the emitted particles. This attribute has no meaning if circle_em_random has been set to true. Syntax:

step   #value

step is the keyword. circle_em_step is deprecated from version 1.2
#value is a real value > 0 that defines the increments of the angle. The smaller the value, the closer the distance between the emitted particles. Default: 0.1.

Angle

The angle defines the start position on the circle. This attribute has no meaning if circle_em_random has been set to true. Syntax:

angle   #value

angle is the keyword. circle_em_angle is deprecated from version 1.2
#value is a real value that defines the start position on the circle. Default: 0.

Emit Random

Particles can be emitted random on the circle, or following the countour of the circle. Syntax:

emit_random   true | false

emit_random is the keyword. circle_em_random is deprecated from version 1.2
true | false defines whether particles are emitted random on the circle (true) or follow the contour. Default: true.

Normal

The normal is used to set the orientation of the circle. Syntax:

normal   #x   #y   #z

normal is the keyword. circle_em_normal is deprecated from version 1.2
#x represents the x-value of the normal. Default value of the normal: Identity.
#y represents the x-value of the normal.
#z represents the x-value of the normal.
