Randomiser

With the Randomiser it is possible to reposition or redirect a particle at a given interval.

Definition

Syntax:

affector   Randomiser   [affectorName]   {affector properties}

See ParticleAffector for explanation of the syntax.

Example: 

affector                        Randomiser
{
   max_deviation_x              10
   max_deviation_y              10
   max_deviation_z              10
   time_step                    0.1
   use_direction           false
}

Properties


Maximum deviation - x value

Defines the x value of the maximum position- or direction change. Syntax:

max_deviation_x   #x

max_deviation_x is the keyword. rand_aff_max_deviation_x is deprecated from version 1.2
#x is the max. x value of the deviation. Default: 1.0.

Maximum deviation - y value

Defines the y value of the maximum position- or direction change. Syntax:

max_deviation_y   #y

max_deviation_y is the keyword. rand_aff_max_deviation_y is deprecated from version 1.2
#y is the max. y value of the deviation. Default: 1.0.

Maximum deviation - z value

Defines the z value of the maximum position- or direction change. Syntax:

max_deviation_z   #z

max_deviation_z is the keyword. rand_aff_max_deviation_z is deprecated from version 1.2
#z is the max. z value of the deviation. Default: 1.0.

Random Direction

By default, the direction of a particle is changed, but it is also possible to let the position change. Syntax:

use_direction   true | false

use_direction is the keyword. rand_aff_direction is deprecated from version 1.2
true | false indicates whether the direction changes (=true) or the position. Default: true.

Time Step

Time Step defines the update interval. After each time step, the particle is repositioned or gets another direction. Syntax:

time_step   #value

time_step is the keyword. rand_aff_time_step is deprecated from version 1.2
#value is a real value that indicates the update interval. Default: 0.
