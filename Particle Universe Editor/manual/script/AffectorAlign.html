Align Affector

Aligns the orientation of a particle towards the previous particle and adjusts the height of the particle (optional), which becomes the length between the two particles. With this combination, it is possible to create a chain of particles (with the right renderer settings).

Definition

Syntax:

affector   Align   [affectorName]   {affector properties}

See ParticleAffector for explanation of the syntax.

Example: 

affector                    Align
{
   resize                   true
}

Properties


Resize

With this attribute it is possible to either adjust the height of a particle related to the previous particle, or not. Syntax:

resize   true | false

resize is the keyword. align_aff_resize is deprecated from version 1.2
true | false indicate whether the particle must be resized (true) or not. Default: false
