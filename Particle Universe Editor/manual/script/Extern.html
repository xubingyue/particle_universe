Extern

Definition

The syntax of a Extern definition is:

extern   externType   [externName]   {extern properties}

extern is the keyword.
externType is the type of extern.
externName defines the name of the extern. The externName is optional but required if the extern must be identified.
The extern properties are handled in the paragraph 'Properties' of each specific Extern.

Example: 

extern                  PhysX
{
   ...
}
