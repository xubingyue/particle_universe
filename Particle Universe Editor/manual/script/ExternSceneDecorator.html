Scene Decorator Extern

The Scene Decorator Extern is more like a test class for fast prototyping. The Scene Decorator Extern makes it possible to add scene objects to the scene so it can be viewed in combination with the particle system. It is possible to add more than one Scene Decorator Extern to a technique. Note, that this is used for testing and prototyping. It has no use in a real application.

Definition

Syntax:

extern   SceneDecorator   [externName]   {extern properties}

See Extern for explanation of the syntax.

Script example: 

system with_an_extern
{
    technique
    {
        ....
        extern                           SceneDecorator
        {
            mesh_name                    Barrel.mesh
            material                     RustyBarrel
            scene_node_scale             150 10 150
            scene_node_position          0 -300 0
        }
    }
}

Properties


Mesh Name

Name of the mesh that must be displayed in the scene. Syntax:

mesh_name   #value

mesh_name is the keyword. scene_mesh_name is deprecated from version 1.2
#value is the name of the mesh.

Material Name

Material name that is applied to the mesh. Syntax:

material   #value

material is the keyword. scene_material_name is deprecated from version 1.2
#value is the material name.

Scale

Scale the node to which the mesh (entity) is attached. The node on its turn is a child of the node to which the particle system is attached. Syntax:

scene_node_scale   #value

scene_node_scale is the keyword.
#value is the scale applied to the node. Default: Vector3::UNIT_SCALE.

Position

Position of the node to which the mesh (entity) is attached. This is relative to the position of node to which the particle system is attached. Syntax:

scene_node_position   #value

scene_node_position is the keyword.
#value is . Default: 1.0.
