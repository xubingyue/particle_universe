Line Emitter

The Line Emitter is used to emit particles on or near a straight line. Because this emitter has several settings, the type of emission changes significantly if certain settings have been choosen. Roughly we distinguish:

- Particles are emitted random on a given line.
- Particles are emitted random on a certain distance from the line.
- Particles are emitted incremental from begin to the end of the line (either on the line or with a certain distance from the line)

Playing with the 'auto_direction', which is a base emitter attribute, and the 'direction' attribute, variations in direction are possible. The 'auto_direction' moves the particles in a direction perpendicular from the line.

Definition

Syntax:

emitter   Line   [emitterName]   {emitter properties}

See ParticleEmitter for explanation of the syntax

Example: 

emitter               Line
{
   ...
   end                100 200 300
   auto_direction     true
}

Properties


Line end

Defines the direction and length of the line by means of a 3d vector. Syntax:

end   #x   #y   #z

end is the keyword. line_em_end is deprecated from version 1.2
#x is the x value of the vector. Default: 0.
#y is the y value of the vector. Default: 0.
#z is the z value of the vector. Default: 0.

Max Increment

Using increments, it is possible to emit particles incremental from begine to the end of the line. This attribute defines the maximum increment. Syntax:

max_increment   #value

max_increment is the keyword. line_em_max_increment is deprecated from version 1.2
#value Is the maximum size of a line segment. Default: 0.

Min Increment

Using increments, it is possible to emit particles incremental from begine to the end of the line. This attribute defines the minimum increment. Syntax:

min_increment   #value

min_increment is the keyword. line_em_min_increment is deprecated from version 1.2
#value Is the minimum size of a line segment. Default: 0.

Max Deviation

Setting the maximum deviation, which defines how far (max.) from the line a particle emits. Syntax:

max_deviation   #value

max_deviation is the keyword. line_em_max_deviation is deprecated from version 1.2
#value is the maximum distance from the line. Default: 0.
