*.pu script

A *.pu script (Particle Universe script) is the actual script in which you define your Particle System. Each Particle System definition in the script is stored by the ParticleSystemManager as a template. New instances of the Particle System can be created using the template as a blueprint for the new Particle System. An example of a Particle System definition using 2 aliasses:

system simplesystem
{
   technique
   {
      // use the billboard renderer defined in the *.pua script
      use_alias                          defaultBillboardRenderer

      material                           ParticleUniverse/Blueball
      default_particle_width             20
      default_particle_height            20
      visual_particle_quota              500

      // use the emitter defined in the *.pua script
      use_alias                          simpleEmitter
      
      affector                           linearForce
      {
         force_aff_vector                0 -100 0
      }
   }
}

Note, that aliasses can also be defined in a *.pu script, but in that case it is not guaranteed that the alias is also parsed before the actual usage of the alias, which would cause an error. That is the reason why it is strongly recommended to define all aliasses in one or more *.pua scripts.
