Create a ParticleSystem

Creation of a Particle Universe Particle System differs slightly from creation of a Particle System using Ogre's ParticleFX plugin. A Particle Universe Particle System is created by means of the ParticleSystemManager, while the Particle System in the ParticleFX situation is created by means of the SceneManger.
A code example how to create a Particle Universe Particle System:

#include "ParticleUniverseSystemManager.h"
...
ParticleUniverse::ParticleSystem mySys = ParticleUniverse::ParticleSystemManager::getSingleton().
    createParticleSystem("mySystem", "systemNameFromScript", sceneManager);
SceneNode* particleNode = sceneManager->getRootSceneNode()->createChildSceneNode();
particleNode->attachObject(mySys);
particleNode->setPosition(Vector3(0, -150, 0));
mySys->prepare();


...
mySys->start ();

Another difference is that the Particle Universe version of the Particle System has a lifecycle that includes : creation, preparing, starting, pausing, resuming, stopping and deletion.

In the example, the Particle System is prepared first, meaning that time consuming initialisations are executed before the Particle System is started. Starting a Particle System is therefor slightly faster. Note, that if the start is called without the prepare first, the initialisations are also executed.

The lifecycle in short:

  1. Create - Create a Particle System
  2. Prepare - Perform initialisation activities, such as creating a particle pool
  3. Start - Start the Particle System
  4. Pause - Pause the Particle System; the particles are still displayed, but do not move (useful for pausing a game)
  5. Resume - Resume the Particle System again; particles start moving again
  6. Stop - Stop the Particle System; particles aren't visible anymore
  7. Delete - Delete the Particle System
