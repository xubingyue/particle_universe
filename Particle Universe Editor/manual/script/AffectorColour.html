Colour Affector

The Colour Affector changes the colour of a particle over its lifetime. During the life of the particle, the colour is interpolated between control points (defined by means of the 'colour_aff_time_colour' attribute). Note, that to use this affector, the material that is used for the particles must have set 'lighting off' in the material pass.

Definition

Syntax:

affector   Colour   [affectorName]   {affector properties}

See ParticleAffector for explanation of the syntax.

Example: 

affector                    Colour
{
   time_colour              0 1 0 0 1
   time_colour              1 0 0 0 1
}

Properties


Time and Colour

Set a certain colour at a given time. Syntax:

time_colour   #timeFraction   #red   #green   #blue   [#alpha]

time_colour is the keyword. Multiple 'colour_aff_time_colour' attributes can be added. colour_aff_time_colour is deprecated from version 1.2
#timeFraction is a real value that defines the the fraction of the time that has passed during the lifetime of the particle. I.e. if the total lifetime of a particle is 10 seconds, and 2.5 seconds have been passed, the timeFraction is 0.25 (2.5 / 10). This means that the timeFraction lies between 0..1
#red defines the red value (between 0..1).
#green defines the green value (between 0..1).
#blue defines the blue value (between 0..1).
#alpha is an optional value and only added when needed (for example if scene_blend alpha_blend is used in the material). Its value lies between 0..1.

Colour Operaton

Defines how the particle affected with the colours defined in 'Time and Colour'. Current operations are
- 'set', which basicly assigns (sets) the colour to a particle.
- 'multiply', which multiplies the colour of the affector on a given time with the colour of the particle.
Syntax:

colour_operation   set | multiply

colour_operation is the keyword.
set means that the colour generated by the affector on a given time is set in the particle.
multiply means that the colour generated by the affector on a given time is multiplied with the colour of the particle.
