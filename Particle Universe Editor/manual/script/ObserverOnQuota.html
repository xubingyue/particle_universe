On Quota Observer

The Particle Universe plugin already identifies several ways to manage the amount of emitted particles. Examples:

  • Using the 'force_emission' attribute in a particle emitter, forces the emitter to emit a number of particles (defined by the emission rate) in one go.
  • With the OnCount Observer the number of emitted particles is counted. One or more handlers are called if a threshold has been exceeded.

The On Quota observer adds another method. This observer determines whether the number of emitted particles has reached the quota. If the 'quota' number of particles has been emitted, one or more event handlers are called. Although it looks the same as the On Count observer, there is a real difference. The On Quota observer validates the number of emitted particles on that particular moment, while the On Count observer just counts the particles over a period of time.

Definition

Syntax:

observer   OnQuota   [observerName]   {observer properties}

See ParticleObserver for explanation of the syntax.

Example: 

observer                   OnQuota
{
   handler                 DoEnableComponent
   {
       enable_component    emitter_component    myEmitter    false
   }
}

Properties

This Observer doesn't have own properties.
