Particle Universe Viewer

The Particle Viewer is a basic viewer application to load, start, pause, resume and stop particle systems. It displays a list of all loaded templates from which yout can pick. Everything works straightforwards, so a detailed description isn´t needed.

Recording

The recording part of the viewer does need some more explanation. If a particle system is selected and the ´Record´ button is pressed, the particle system starts and a ´video´ recording is made. The output is a list of image files, where every image represents one frame. Stitching them all to a real video format is not supported (you should use a video editing tool for that), but it is possible to create one atlas image from them with the Atlas Image Tool. This atlas image can be played with the use of a the Texture Animator

Recording isn´t done in realtime, so that explains why a setting of 60 frames per minute takes longer than 1 minute. The recording settings are included in the puviewer.cfg. This configuration file includes the following settings:

VideoPath is a (relative)directory that defines where the recorded image files are saved.
ImageWidth defines the width of each image file.
ImageHeight defines the height of each image file.
FramesPerMinute defines the number of frames per minute.
FileNameSuffix defines the extension of the image files (".png" for example).
StartTime defines how many seconds the recording waits after the Record buttons has been pressed.
EndTime defines how many seconds it takes to stop recording.
PauseTime defines the pause after each image file (frame) has been saved. The pause time is needed for calibration.
