Sine Force Affector

The Sine Force Affector applies a non-linear force to a particle. Using a sine-function, the force varies and gives a wave effect on the movement of the particle.

Definition

Syntax:

affector   SineForce   [affectorName]   {affector properties}

See ParticleAffector for explanation of the syntax.

Example: 

affector                      SineForce
{
   force_vector               30 0 0
   frequency_min              0
   frequency_max              3
}

Properties


Force Vector

Defines the direction to which the force applies. Syntax:

force_vector   #x #y #z

force_vector is the keyword. force_aff_vector is deprecated from version 1.2
#x is a real value that defines the x coordinate of the force vector. Default: 0.
#y is a real value that defines the y coordinate of the force vector. Default: 0.
#z is a real value that defines the z coordinate of the force vector. Default: 0.

Force Application

Defines how the force vector is applied. Syntax:

force_application   add | average

force_application is the keyword. force_aff_application is deprecated from version 1.2
add | average. The 'add' value adds a fraction of the force vector to the particles' direction vector, which causes the particles' movement being affected. In case of average, an average value of the particles' direction and the force vector is calculated. Default: add.

Minimum Frequency

A sine wave with a frequency that lies between a minimum and maximum frequency is generated and multiplied with the force factor. This gives the particles a waving motion. A new frequency is determined after every sine cycle. This property defines the minimum frequency. Syntax:

frequency_min   #value

frequency_min is the keyword. sinef_aff_frequency_min is deprecated from version 1.2
#value is a real value that defines the minimum frequency of the sine. Default: 1.0.

Maximum Frequency

This property defines the maximum frequency (see also frequency_min). Syntax:

frequency_max   #value

frequency_max is the keyword. sinef_aff_frequency_max is deprecated from version 1.2
#value is a real value that defines the maximum frequency of the sine. Default: 1.0.
