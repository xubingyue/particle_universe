On Position Observer

Observes a particle and calls the registered event handlers as soon as the particle has a certain (world)position. For instance, if the particles' position is (100, -100, 200) and the observer validation is set to "position_y less_than 0", the validation is true and the event handlers are called.

Definition

Syntax:

observer   OnPosition   [observerName]   {observer properties}

See ParticleObserver for explanation of the syntax.

Example: 

observer                OnPosition
{
   position_y           less_than    0
   handler              DoExpire {}
}

Properties


Position X

Set the x-threshold. If the x-coordinate (position) of the particle is equal, less or greater than this value, the event handlers are triggered. Syntax:

position_x   less_than | greater_than | equals   #value

position_x is the keyword.
less_than | greater_than | equals defines the comparision operator. Default: less_than
#value is the threshold value that determines when the event handlers are triggered. Default: 0
