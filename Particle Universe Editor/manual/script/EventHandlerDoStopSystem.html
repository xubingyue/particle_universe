Do Stop System Event Handler

This event handler stops the whole particle system as soon it is called by the particle observer at which this event handler is registered.

Definition

Syntax:

handler   DoStopSystem   [handlerName]   {}

See ParticleEventHandler for explanation of the syntax.

Example: 

handler                 DoStopSystem {}

Properties

This Event Handler doesn't have properties.
