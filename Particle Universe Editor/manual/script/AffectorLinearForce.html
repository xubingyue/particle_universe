Linear Force Affector

The Linear Force Affector applies a constant force to a particle. This 'forces' the particle to a certain direction.

Definition

Syntax:

affector   LinearForce   [affectorName]   {affector properties}

See ParticleAffector for explanation of the syntax.

Example: 

affector               LinearForce
{
   force_vector        0 -100 0
}

Properties


Force Vector

Defines the direction to which the force applies. Syntax:

force_vector   #x #y #z

force_vector is the keyword. force_aff_vector is deprecated from version 1.2
#x is a real value that defines the x coordinate of the force vector. Default: 0.
#y is a real value that defines the y coordinate of the force vector. Default: 0.
#z is a real value that defines the z coordinate of the force vector. Default: 0.

Force Application

Defines how the force vector is applied. Syntax:

force_application   add | average

force_application is the keyword. force_aff_application is deprecated from version 1.2
add | average. The 'add' value adds a fraction of the force vector to the particles' direction vector, which causes the particles' movement being affected. In case of average, an average value of the particles' direction and the force vector is calculated. Default: add.
