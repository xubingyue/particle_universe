Plane Collider

The Plane Collider represents a plane shape with which the particles collide.

Beware that the Plane Collider is part of Particle Universe itself. If you want more sophisticated collision and responses, Particle Universe also has the option to integrate NvidiaÂ´s PhysX engine, formerly owned by Ageia.

Definition

Syntax:

affector   PlaneCollider   [affectorName]   {affector properties}

See ParticleAffector for explanation of the syntax.

Example: 

affector                     PlaneCollider
{
    position                 0 -100 0
    normal                   0 1 0
    intersection             point
    collision_type           bounce
    bouncyness               0.4
}

Properties


Plane Normal

Defines the normal vector of the plane. Syntax:

normal   #x   #y   #z

normal is the keyword. plane_collider_normal is deprecated from version 1.2
#x is the x value of the normal vector. Default: 0.
#y is the y value of the normal vector. Default: 0.
#z is the z value of the normal vector. Default: 0.
