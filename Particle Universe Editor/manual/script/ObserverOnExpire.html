On Expire Observer

Observes a particle and calls the registered event handlers as soon as the particle is exipred.

Definition

Syntax:

observer   OnExpire   [observerName]   {observer properties}

See ParticleObserver for explanation of the syntax.

Example: 

observer                OnExpire
{
   ...
}

Properties

This Observer doesn't have own properties.
