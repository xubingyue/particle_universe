Box Collider Extern

Look at Sphere Collider for a description. The Box Collider behaves the same with the difference that it is a box.

Definition

Syntax:

extern   BoxCollider   [externName]   {extern properties}

See Extern for explanation of the syntax.

Script example: 

system with_an_extern
{
    technique
    {
        ....
        extern                        BoxCollider
        {
            box_width                 300
            box_height                300
            box_depth                 300
            intersection              point
            collision_type            bounce
            bouncyness                1
        }
    }
}

Additional code example to attach the extern to a SceneNode: 

SceneNode* externNode = mSceneMgr->getRootSceneNode()->createChildSceneNode();
Attachable* attachable = static_cast<Attachable*>(sys->getTechnique(0)->getExtern(0));
if (attachable)
{
    externNode->attachObject(attachable);
    externNode->setPosition(Vector3(-300, 100, -100));
}

Properties


Distance Threshold

Maximum distance where the collision detection of the Extern Box Collider isn't performed anymore. Syntax:

distance_threshold   #value

distance_threshold is the keyword. attachable_distance_threshold is deprecated from version 1.2
#value is a real value that defines the threshold speed. Default: Infinite.
